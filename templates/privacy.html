Privacy Policy

  1. Lopad pads won't be republished elsewhere but can be viewed by anyone with access to the URL.
  2. Details of your access to the server will be logged for the purpose of maintaining the service.
  3. We won't look at the content of pads unless absolutely necessary for the continuing safe operation of our service.
  4. We will release logs to authorised officials if we are legally obliged to do so.
  5. Logs will not be released under any other circumstances.
  6. This privacy policy is subject to change without notice. The current version can be found at http://www.lopad.org/privacy.html.

Cookies

Lopad must set a number of cookies in order to function. You must accept these cookies in order to use Lopad. Each cookie is described below.

express_sid
Session cookie
Expires at end of session
Uniquely identifies each user collaborating on a pad
prefs
Session cookie
Expires never
Stores pad editor settings
token
Random string
Identifies the user as a pad author
Expires after 60 days
